¿Qué es el Frontend?

El frontend es la parte del desarrollo web que se enfoca en la creación de interfaces y experiencias de usuario en un sitio web. Utiliza tecnologías como HTML, CSS y JavaScript para construir elementos visuales con los que los usuarios interactúan.
